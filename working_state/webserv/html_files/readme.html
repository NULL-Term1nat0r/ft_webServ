How to use this Webserv:

"File Upload" will direct you to the page where you can upload files. Simply choose a file by pressing "Choose a file", and upon making your selection, upload the file by pressing "Upload File" button.

If you wish to view all the files that you have uploaded, simply press the "Uploaded picture files" button, and you will be redirected to the page where all your files are displayed.

"Cookie Test" will direct you to the page where you can verify that the bonus of the subject is working. Simply press the "Save Client IP" button, and you will create a cookie.

"Cgi Test" will direct you to the page where you can test the CGI scripts. Simply enter your name in the text box and press "Submit" button. You will be redirected to the page where you can see the result of the CGI script.

"Creators" will direct you to the page where you can find the intra names of the creators and their GitHub links.

"Readme" button is self-explanatory, and you are already using it, so stop wasting your time and get to testing!
